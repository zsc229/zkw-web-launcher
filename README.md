# 🚀 zkw-web-launcher (zkw 工作室 Web 版开服器)

> **一款基于 Flask 的现代化 Minecraft 服务器 Web 管理平台。**
>
> 🌟 **核心亮点**：Web 界面随时随地管理 | 内置账号注册/登录系统 | 积分经济体系 (签到/兑换/续费) | 实时日志控制台 | 一键上传核心 | 自动环境检测
>需要安装python，不能在手机上运行。
---

## 📖 项目简介

**zkw-web-launcher** 彻底改变了传统 MC 开服器需要下载客户端、配置本地环境的繁琐模式。
只需在浏览器输入地址，即可实现**全平台、远程、可视化**的服务器管理。无论是个人服主还是小型腐竹团队，都能通过它轻松搭建属于自己的私有云服平台。

### ✨ 核心功能

*   **🌐 纯 Web 架构**
    *   无需下载客户端，浏览器即可访问。
    *   支持手机、平板、电脑多端适配。
    *   局域网/公网均可部署，随时随地管理服务器。

*   **🔐 完整的账号体系**
    *   支持用户**注册/登录**，数据本地加密存储 (SHA-256)。
    *   **管理员特权**：内置超级管理员，拥有无限积分和最高权限。
    *   **安全机制**：密码哈希加密，防止明文泄露。
 
    *   *   **💰 独特的积分经济系统**
    *   **每日签到**：用户每日登录可领取随机积分 (100-500 分)。
    *   **兑换码功能**：支持输入兑换码获取积分 (适合运营活动)。
    *   **服务器续费**：`10 积分 = 1 天` 使用时长的模拟计费逻辑，过期自动禁止启动。

*   **🛠️ 强大的服务器管理**
    *   **一键上传核心**：支持网页端直接上传 `.jar` 核心文件。
    *   **实时控制台**：WebSocket/SSE 技术推送实时日志，支持在线发送指令。
    *   **多核心管理**：支持上传、切换、删除多个服务端核心。
